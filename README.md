# TeristaSpace

A modernized and secure Android virtualization framework for personal use.

## Features

- **Modern Android Support**: Full compatibility with Android 12-16 (API 31-35)
- **Advanced Virtualization**: Run multiple isolated app instances
- **Privacy-First**: No tracking, analytics, or telemetry
- **Security Enhanced**: Modern permissions, secure storage, clean code
- **Optimized Performance**: Updated dependencies and refined architecture

## Build Requirements

- Android Studio Hedgehog | 2023.1.1 or later
- Gradle 8.11.1
- Android Gradle Plugin 8.7.3
- Kotlin 2.1.0
- JDK 21
- Android SDK API 35
- NDK 29.0.13846066

## Building

1. Clone the repository
2. Open in Android Studio
3. Sync project
4. Build → Generate Signed Bundle/APK

## Package Structure

- `com.terista.space` - Main application
- `com.terista.space.core` - Core virtualization framework
- `com.terista.space.reflection` - Reflection utilities
- `com.terista.space.compiler` - Compile-time processing
